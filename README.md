A prototype of a platformer game which uses grapple hook mechanics.

**How to Run**
- Use the itch.io link here: https://omeletted.itch.io/grappleformer
- Download the repo and open it in the Godot Engine, and you can edit/export it however you prefer.
- The controls are the arrow keys and the X and C keys
